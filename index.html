WebGIS Mount Tambora Trekking Routes (Final Project)

A web-based Geographic Information System (GIS) that presents the trekking routes of Mount Tambora interactively. This project aims to provide spatial and non-spatial information to assist hikers in planning their routes. Flood-Prone Area Mapping (Internship Project)

This project aims to identify and map flood-prone areas in Central Lombok Regency using Geographic Information System (GIS) techniques, utilizing overlay methods to analyze and combine various data. The goal is to provide an accurate spatial-based flood risk map to support disaster preparedness and urban planning.

WebGIS Interactive Village Map (Community Service Project)

This project aims to create an Interactive Village Map that presents information about important locations within the village. Using ArcGIS Online and Survey123, this map facilitates the presentation of essential geospatial data, such as public facilities, infrastructure, and other important locations within the village. Data collection is conducted using the Survey123 app, which enables efficient and accurate field surveys.
